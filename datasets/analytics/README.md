# Analytics Dataset (Gold Layer)

This folder represents the final processed datasets used for analysis.

## Files:
- gold.dim_customers.csv
- gold.dim_products.csv
- gold.fact_sales.csv

## Description:
These datasets are derived from the Gold Layer of the Data Warehouse after data cleaning and transformation.

They are used for:
- Exploratory Data Analysis (EDA)
- Advanced Analytics
- Reporting
